Embedded Coder Support Package for Texas Instruments AM26X Processors

Version: 24.2.1

Design models in Simulink, generate code using Embedded Coder, and run the executables on AM26x ControlCard and LaunchPad, platforms designed for real-time control applications without using manual programming. Apply industry-proven techniques for Model-Based Design to verify that the algorithms work during simulation and then implement the algorithms as standalone applications on the AM26X platforms using automatic code generation. By default, the generated code is ANSI/ISO C/C++.
Embedded Coder Support Package for Texas Instruments AM26X Processors includes:
  • Library of Simulink blocks that connect to AM26x CC and LP I/O such as
  • ADC, DAC, CMPSS
  • EPWM, ECAP, EQEP, SDFM
  • GPIO,
  • UART, MCAN, SPI
  • HWI
  • Model deployment for standalone operation
  • Built-in support for specific hardware, scheduler and compiler
  • PIL, External mode (XCP over Serial, XCP over CAN)
