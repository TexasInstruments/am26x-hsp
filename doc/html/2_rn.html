Known issues

ID
Issue
MCUSDK-13801
Package missing .p files needed by FreeRTOS.
The missing .p files are:
1) FreeRTOSConfigurator.p
2) getModelRates.p
3) getShrdCompRootDir.p
4) onAfterCodegenHookFreeRTOS.p
5) onFreeRTOSSelect.p
Workaround: Please install the Embedded Coder package for ARM Cortex R and pick these files from that package. Location to add these files in EC package for AM26x:
FreeRTOSConfigurator.p - $(TARGET_ROOT)/+freeRTOS/+matlabshared/+freertos/
All the other .p files mentioned above should be placed in below directory:
$(TARGET_ROOT)/+freeRTOS/+matlabshared/+freertos/+internal
