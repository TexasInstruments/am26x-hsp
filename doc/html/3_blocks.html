ADC

Analog to Digital Converter (ADC)

Description

The ADC block performs analog-to-digital conversion of the input signal configured in selected ADC pins. The output of ADC block is digital values corresponding to input analog signals, this output is stored in the ADC result registers. This block can be used in cases that requires analog values from external sources to be digitalized.

Ports

Output

The output of the AM263x ADC is uint16 value which is stored in an ADC result register, and the ouput values are in the range of 0 to 4095 because the ADC is a 12-bit converter.

Parameters

Module

This parameter specifies which ADC module should be used among the 5 Analog-to-Digital Converter modules integrated in the device.

Prescaler

This parameter specifies the prescaler value that determines the ADC clock, this ADC clock is used to clock the converter.

Resolution Mode:

This parameter specifies the resolution of the converter.

SOC Priority:

This parameter configures the priority mode of the SOCs. There are three main modes that can be passed to this parameter:
  • All SOCs round-robin mode, so no SOC has a higher priority than other
  • All SOCs in high priority mode, where the priority of the SOC is determined by SOC number
  • A range of SOCs are assigned high priority with others in round robin mode

Signal Mode

This parameter configures the analog-to-digital converter signal mode:
Sample on single pin with VREFLO - In this mode the input vltage is sampled on a single pin referenced to VREFLO.
Sample on pair of pins - In this mode, the input voltage to the converter is sampled on a pair of input pins, a positive and a negative pin.

SOC Trigger Number

Specify the type of signal that triggers conversion through start-of-conversions. Each SOC is configured to define single conversion of a single channel, there are a total of 16 SOCs

SOCx Acquisition Window

This value specifies the sample window for this SOC, this is the amount of time the sampling capacitor is allowed to charge and is configurable for each SOCx.

SOCx Trigger Source

This value specifies the trigger source for the configured SOC. The values for this can be selected between
  • Software only
  • CPU Timers 0-7
  • GPIO: Input X-Bar INPUT5
  • ADCSOCA or ADCSOCB from each ePWM module

SOC Interrupt Trigger

Select the interrupt trigger source for this SOC, this signal is ORed with SOCx Trigger Source.

Sample Time

Determines the interval at which the block reads data, the time between consecutive sets of samples that are converted for selected ADC channels.

Data Type

Select the data type of the output data

Enable ADC Converter

Setting this parameter powers up the analog-to-digital converter module.

Pulse Mode

This parameter sets the timing of the end-of-conversion pulse. This end of conversion pulse will be generated either at the end of the acquisition window or at the end of the voltage conversion.

Post Interrupt at EOC Trigger

Enabling this parameter will allow user to configure an ADC Interrupt.

Interrupt Selection

This parameter allows the user to configure any of the available ADC Interrupts, ADC_INT_NUMBER1, ADC_INT_NUMBER2, ADC_INT_NUMBER3, or ADC_INT_NUMBER4.

Interrupt Source

This parameter sets which SOC conversion is the source of an ADC interrupt. This parameter has input values ranging SOC0-SOC15

Enable Continuous Mode

Enable continuous mode for ADC interrupt, in this mode pulses will be generated for the specified ADC interrupt whenever an EOC pulse is generated irrespective of whether or not the flag bit is set.

Conversion Channels

Configures the ADC channel whose input needs to be converted by the selected SOC. The conversion channels listed is dependent on signal modes configured. If the signal mode is configured to sample on single pin, then SOC selects a single channel to be treated as input. If the signal mode is configured for sampling on a pair of pins, then the SOC selects an odd-even pair of channel as input.

PPB Block

Select one of the four avaliable post-processing blocks available, which is associated with ADC result registers

PPB SOC

Select the SOC for this PPB

PPB Calibration Offset

This specifies the 10-bit signed value that gets subtracted from ADC output, so this sets the PPB offset correction value.

PPB Reference Offset

This parameter specifies the 12-bit unsogned value that gets subtracted from ADC output. This is used to calculate feedback error by subtracting a reference value.

PPB High Trip Limit

This parameter sets trip high limit for digital comparator, thereby setting the windowed trip limits for a PPB. These values set the digital comparator so that when one of the values is exceeded, either a high or low trip event will occur.

PPB Low Trip Limit

This parameter sets trip low limit for digital comparator, thereby setting the windowed trip limits for a PPB. These values set the digital comparator so that when one of the values is exceeded, either a high or low trip event will occur.

PPB Enable Event : ADC Event Trip High

Setting this parameter will enable trip high event to source a PPB event. This will allow trip high event to propogate through the X-BAR to a pin or to an ePWM module.

PPB Enable Event : ADC Event Trip Low

Setting this parameter will enable trip low event to source a PPB event. This will allow trip low event to propogate through the X-BAR to a pin or to an ePWM module.

PPB Enable Event : ADC Event Zero Crossing

Setting this parameter will enable zero crossing event to source a PPB event. This will allow zero crossing event to propogate through the X-BAR to a pin or to an ePWM module.

PPB Enable Interrupt Event : ADC Event Trip High

Enabling this parameter sets ADC Trip High event as an ADC PPB interrupt source. Only the sources that are enabled can be reflected to the processor interrupt.

PPB Enable Interrupt Event : ADC Event Trip Low

Enabling this parameter sets ADC Trip Low event as an ADC PPB interrupt source. Only the sources that are enabled can be reflected to the processor interrupt.

PPB Enable Interrupt Event : ADC Event Zero Crossing

Enabling this parameter sets ADC Zero crossing event as an ADC PPB interrupt source. Only the sources that are enabled can be reflected to the processor interrupt.

PPB Enable Two's complement

This parameter enables two's complement capability in PPB. When this is enabled a two's complement will be performed on the output of the offset subtraction before it is stored in the appropriate PPB result register. So ADCPPB Result = (ADC REFERENCE OFFSET - ADC RESULT)

PPB Enable CBC clear event

This parameter enables cycle-by-cycle clear of ADC PPB event flags, so the desired PPB event flags are automatically cleared on the next PPB Result load.

Burst Mode Trigger signal

This selects the trigger signal for the ADC burst mode, which allows this single trigger to walk through the round-robin SOCs one or more at a time.

Burst Size

The size of the burst for the ADC burst mode, this determines which round-robin SOC will be set on reception of burst trigger.

CMPSS

Comparator Subsystems (CMPSS)

Description

The comparator subsystems consists of two modules, Comparator High (COMPH) and Comparator Low (COMPL). Each CMPSS module is designed around a pair of analog comparators which generates a digital output indicating if the voltage on the positive input is greater than the voltage on the negative input. The output will be high if the input on the positive pin is greater than the input on the negative pin.

Ports

Output

The output port STS, gives the comparator status.
0 - If STS is 0, then voltage on positive pin is lesser than voltage on negative pin
1 - If STS is 1, then voltage on positive pin is greater than the voltage on negative pin

ECAP

Enhanced Capture Module

Description

The enhanced capture module allows time-based logging of external signal transitions. It is used to accurately time external events by time stamping transitions on the capture input pin. ECAP can be used to measure the speed of a rotating machine, determine the elapsed time between pulses, calculate the period and duty cycle of a pulse train signal, and decode current/voltage measurements derived from duty cycle encoded current/voltage sensors.

Ports

Input

T - This input port specifies the APWM period. Port appears only when the ECAP Mode is APWM and when Specify APWM Period is Specify via Input Port
D - This input port specifies the APWM compare value. Port appears only when the ECAP Mode is APWM and when Specify APWM Period is Specify via Input Port
R - This input port determines if the ECAP will be re-armed. Port appears only when the ECAP Mode is Capture mode, the Capture Mode is One shot capture and when Re-arm ECAP is enabled.

Output

TS - This is the output time stamp of capture events. TS is a variable size vector where the size depends on the Capture stop at event value.
CF - This output port provides the status of the capture event. CF is a variable size vector where the size depends on the Capture stop at event value. 0 indicates no event has occured, 1 indicates that the event specified by the Capture stop at event parameter has occured on the ECAP pin. This port appears only when Enable capture event status flag is checked.
OF - This output port provides the status of overflow. 0 indicates counter has not overflowed. 1 indicates that the counter has overflowed.This port is available only when the Enable overflow status flag is checked.

EPWM

Enhanced Pulse Width Modulator (EPWM)

Description

This block configures the Enhanced Pulse Width Modulator(EPWM) to generate PWM waveforms. There are in a total of 32 EPWM modules supported by AM263x, and each of the EPWM module generates two PWM signals ePWMA and ePWMB

Time base clock prescaler divider

The time base clock precsaler divide is used to configure the Time-base clock speed (TBCLK) of the configured ePWM module. Calculate TBCLK using the equation:
TBCLK in Hz = PWM clock in Hz/(HSPCLKDIV * CLKDIV)
TBCLK in seconds = 1/TBCLK in Hz = 1/100 MHz = 0.01 microseconds
The choices for Time base clock Prescaler Divider are : 1, 2, 4, 8. 16. 32. 64. 128

High Speed time base clock prescaler divider

The high speed time base clock prescaler divider is used to configure the Time-base clock speed (TBCLK) of the configured ePWM module. Calculate TBCLK using the equation:
TBCLK in Hz = PWM clock in Hz/(HSPCLKDIV * CLKDIV)
TBCLK in seconds = 1/TBCLK in Hz = 1/100 MHz = 0.01 microseconds
The choices for High Speed Time Base Clock Prescaler Divider are : 1, 2, 4, 6, 8, 10, 12, 14
